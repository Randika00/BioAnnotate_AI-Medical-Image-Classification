# BioAnnotate_AI-Medical-Image-Classification

BioAnnotate_AI is a practical Artificial Intelligence / Machine Learning (AI/ML) project developed to support the annotation, preprocessing, and classification of medical and biomedical image datasets.
This project is designed to organize image data into structured categories such as Age, Anatomy, Genus, Number, Sex, Skin, and Type, making the dataset ready for machine learning and deep learning workflows. The goal of BioAnnotate_AI is to build a scalable AI/ML medical image annotation framework that can later be extended into a complete intelligent classification system for disease-related and biomedical visual analysis. 

The system includes image preprocessing operations such as:

    ∗ image loading
    ∗ category labeling
    ∗ category labeling
    ∗ numerical encoding
    ∗ .npy training file generation

The main objective of this project is to build a strong foundation for Convolutional Neural Network (CNN) training and future automated image classification tasks in healthcare and biomedical research domains. The project is still under active development, with plans to train and evaluate CNN models for each image category and incorporate data augmentation to enhance model performance. Further improvements include leveraging advanced transfer learning models such as VGG16, ResNet, or EfficientNet to improve classification accuracy, as well as building a user-friendly annotation dashboard and integrating model predictions into a real-time annotation workflow.
This project demonstrates my practical experience in:

    ∗ AI/ML workflow design
    ∗ medical image dataset preparation
    ∗ computer vision preprocessing
    ∗ deep learning data pipeline development
    ∗ annotation-based model preparation
